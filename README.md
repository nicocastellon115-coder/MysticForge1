# MysticForge — API + SQLite + Seguridad + Backups + Monitorización

## Qué es
API Node.js para gestionar amuletos (demo ASIR). Incluye:
- Base de datos SQLite
- Seguridad: Helmet, CORS restringido, rate-limit, validación y API key
- Logs: consola + `access.log`
- Backups automáticos (Programador de tareas)
- Monitorización externa (Serveo + UptimeRobot)

npm install
